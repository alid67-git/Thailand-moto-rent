# Languages — Thailand Moto Rent

## Supported locales

| Code | Language | Direction | Default |
|------|----------|-----------|---------|
| `tr` | Türkçe | LTR | **Yes** |
| `en` | English | LTR | |
| `de` | Deutsch | LTR | |
| `fr` | Français | LTR | |
| `ar` | العربية | RTL | |

## Default

- **Site default:** Turkish (`tr`)
- **Your work language:** Turkish — all new copy should be added to `src/i18n/messages/tr.ts` first, then mirrored in `en`, `de`, `fr`, `ar`.

## How it works

- Translations live in `src/i18n/messages/*.ts`
- `LocaleProvider` stores choice in `localStorage` (`tmr-locale`)
- Header language dropdown switches UI instantly
- Arabic sets `dir="rtl"` on `<html>`

## Launch geography

- **Active (Phuket):** Patong, Kamala, Kata, Karon
- **Coming soon:** Chiang Mai (shown disabled in area dropdown)

## Adding a new string

1. Add key to `tr.ts`
2. Add same key to `en.ts`, `de.ts`, `fr.ts`, `ar.ts`
3. Use `t("your.key")` in components via `useLocale()`
