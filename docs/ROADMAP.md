# Roadmap — Thailand Moto Rent

## Phase 0 — Foundation ✅

- [x] Separate project folder
- [x] Next.js + TypeScript skeleton
- [x] Domain types (Partner, Bike, Booking)
- [x] Platform rules document
- [x] Customer-facing site (corporate booking flow)
- [x] Fleet: Honda Click 160, ADV 160, Forza 350
- [x] Insurance tiers, top case, cancellation insurance at checkout
- [x] Live price summary on `/book`
- [x] Launch location: Phuket Patong
- [x] i18n: TR (default), EN, DE, FR, AR
- [x] Motorcycle photos (Wikimedia Commons)

## Phase 1 — MVP (Bangkok / Chiang Mai)

- [ ] Database schema & API routes
- [ ] Customer registration & login
- [ ] Partner application & admin approval
- [ ] Bike CRUD for partners
- [ ] Search by city + dates
- [ ] Booking request & confirmation
- [ ] Email notifications

## Phase 2 — Payments & trust

- [ ] Online payment (deposit + rental fee)
- [ ] Partner payout dashboard
- [ ] Reviews & ratings
- [ ] Dispute handling workflow
- [ ] Multi-language (EN / TH)

## Phase 3 — Scale

- [ ] Mobile app (React Native or PWA)
- [ ] More cities (Phuket, Pattaya, Krabi…)
- [ ] Live availability sync
- [ ] Insurance partner integrations
- [ ] Analytics for partners

## Out of scope (for now)

- Real-time GPS tracking (Uber-style dispatch)
- Fleet ownership by platform
- Integration with Cudy or other unrelated projects
